# Логотипы Klust

## Расположение файлов

Разместите файлы логотипов в этой папке: `public/images/logo/`

## Структура файлов

```
public/images/logo/
├── icon.svg          # Иконка (просто буква) - квадратное изображение
├── icon.png          # Альтернативный формат иконки
├── logo-full.svg     # Длинное лого (весь текст Klust) - горизонтальное изображение
└── logo-full.png     # Альтернативный формат длинного лого
```

## Рекомендуемые размеры

- **Иконка (icon)**: 
  - Формат: SVG или PNG
  - Размер: 64x64px или 128x128px (для ретина)
  - Квадратное изображение с буквой/символом

- **Длинное лого (logo-full)**:
  - Формат: SVG или PNG
  - Размер: примерно 200x40px (ширина больше высоты)
  - Горизонтальное изображение с полным текстом "Klust"

## Использование в компонентах

### BaseHeader (автоматически используется во всех Layouts)

Компонент автоматически определяет приоритет:
1. Если указан `logoIcon` - используется иконка
2. Если указан `logoImage` (но не `logoIcon`) - используется длинное лого
3. Если ничего не указано - используется текстовое лого

### Примеры использования

#### В Layouts:

```vue
<ClientLayout
  logo-icon="/images/logo/icon.svg"
  logo-image="/images/logo/logo-full.svg"
/>
```

#### Прямо в BaseHeader:

```vue
<BaseHeader
  logo-text="Кластер"
  logo-icon="/images/logo/icon.svg"
  logo-image="/images/logo/logo-full.svg"
/>
```

#### В GuestLayout:

```vue
<GuestLayout
  title="Вход"
  logo-icon="/images/logo/icon.svg"
  logo-image="/images/logo/logo-full.svg"
/>
```

## Текущие настройки по умолчанию

- **Иконка**: `/images/logo/icon.svg`
- **Длинное лого**: `/images/logo/logo-full.svg`

Если файлы не найдены, автоматически будет использоваться текстовое лого "Klust".

## Форматы файлов

Рекомендуется использовать SVG для лучшего качества на всех устройствах, но поддерживаются также PNG, JPG, WebP.
