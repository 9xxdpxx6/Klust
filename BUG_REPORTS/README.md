# Отчеты о багах (Bug Reports)

Эта директория содержит отчеты о багах, сгруппированные по логическим модулям. Каждый файл описывает баги в соответствующей области приложения.

## Структура

- `01-AUTHENTICATION-USER-MANAGEMENT.md` - Баги аутентификации и управления пользователями
- `02-CASE-MANAGEMENT.md` - Баги управления кейсами
- `03-PARTNER-PORTAL.md` - Баги партнерского портала
- `04-UI-COMPONENTS.md` - Баги UI компонентов и уведомлений
- `05-FILTERS-STATISTICS.md` - Баги фильтров и статистики

## Формат описания бага

Каждый баг содержит:
- **Название**: Краткое описание проблемы
- **Приоритет**: Critical / High / Medium / Low
- **Модуль**: К какой части системы относится
- **Описание**: Подробное описание проблемы
- **Шаги для воспроизведения**: Пошаговая инструкция
- **Ожидаемое поведение**: Как должно работать
- **Фактическое поведение**: Как работает сейчас
- **Файлы для проверки**: Список файлов, которые нужно проверить
- **Предполагаемая причина**: Где может быть проблема
- **Решение**: Рекомендации по исправлению

## Использование

Эти файлы предназначены для:
1. Ревью кода LLM-моделями
2. Отслеживания багов
3. Планирования исправлений
4. Документирования известных проблем
